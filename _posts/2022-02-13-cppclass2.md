C++의 함수에 대해 배워보자.

함수
===
***

프로그래밍을 하면 같은 코드를 여러번 쓰는 경우가 있고, 일일이 코드를 다 적다보면 코드가 엄청나게 길어지게 된다.

함수는 그 문제를 해결하기 위해 존재하며, 같은 코드를 간단하게 실행시키는 역할을 한다.

예시
---
***
```c++
#include<iostream>
int sum(int a;int b){
    return a+b;
}
int main(){
    int a,b,c;
    cin >> a >> b >> c ;
    cout << sum(sum(a+b),c) ;
    }
```
위의 코드는 ```sum``` 함수를 두번 써서 ```a```,```b```,```c```를 입력받고 셋의 합을 출력하는 코드이다.
